# IBU-061
This repository covers my lab work on International Burch University course Fundamentals of Web Development.

Foundations of Frontend Web Development is a comprehensive course designed for beginners seeking to establish a strong foothold in the dynamic realm of web development. Through a structured curriculum, participants will delve into the fundamentals of HTML5, CSS3, and JavaScript, gaining proficiency in building and styling web pages. The course extends into essential concepts such as DOM manipulation, and responsive design, providing a well-rounded skill set.
